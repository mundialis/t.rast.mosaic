DESCRIPTION

t.rast.mosaic is a GRASS GIS addon Python script to remove clouds and tries to fill raster gaps using r.series and r.series.lwr and aggregates temporally the maps of a space time raster dataset by a user defined granularity using t.rast.aggregate.

EXAMPLE

t.rast.mosaic input=s2_B02_10m output=B02_mosaic clouds=s2clouds cloudbuffer=10 \
  granularity='1 months' method=median

SEE ALSO

r.series, r.series.lwr, r.patch, t.rast.aggregate, r.mapcalc
